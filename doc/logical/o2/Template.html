Class o2.Template

static class o2.Template

A really fast template engine.

Defined in template.core

Function Summary
static parse (Object data, String tpl)

Parses the given template.

Usage example:

 var data = {
      users : [
          {name : 'Joe' },
          {name : 'Jill'},
          {name : 'Jack'}
      ]
 };

 var tpl = [
      '
    ',
          ['each users',
              '
  • {{name}}
    • '],
      '
'
 ];

 var html = o2.Template.parse(data, tpl);
